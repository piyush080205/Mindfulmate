# MindfulMate

MindfulMate is a Next.js application that provides a supportive AI chatbot for mental well-being. It offers a safe space for users to express themselves, track their mood, and access mindfulness exercises and helpful information.

## About The Project

This project is designed to be an AI-powered companion that assists users in managing their emotional health. It features:

*   **AI Chat:** A conversational agent for users to talk about their feelings.
*   **Mood Logging:** A simple tool to track daily moods.
*   **Mindfulness Exercises:** Guided text-based exercises for grounding and calm.
*   **FAQs:** A curated list of answers to common mental health questions.

## Getting Started

Follow these instructions to set up the project on your local machine for development and testing.

### Prerequisites

You need to have Node.js and npm (Node Package Manager) installed on your system. You can download them from [nodejs.org](https://nodejs.org/).

### Installation

1.  Clone the repository to your local machine.
2.  Navigate to the project directory in your terminal.
3.  Install the required npm packages:
    ```sh
    npm install
    ```

### Running the Application

1.  To run the development server, execute the following command:
    ```sh
    npm run dev
    ```
2.  Once the server is running, open your browser and navigate to [http://localhost:9002](http://localhost:9002).

The application also includes a Genkit AI flow. To run the Genkit development service separately, you can use:
```sh
npm run genkit:dev
```
