# **App Name**: MindfulMate

## Core Features:

- Free-Form Chat: Engage in open-ended conversations with the AI to express thoughts and feelings.
- Mood Logging: Record daily mood entries using a simple and intuitive interface.
- Personalized Suggestions: Receive AI-driven, tailored coping mechanisms based on mood logs and conversations. The AI acts as a tool by reasoning on if, or when to present supporting mental health information.
- Guided Mindfulness: Access a library of guided mindfulness exercises.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) to evoke a sense of calm and growth.
- Background color: Warm gray (#F0F0F0) to provide a neutral and comforting backdrop.
- Accent color: Muted yellow (#E4C8AF) for highlighting key elements without being jarring.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern but slightly warm look.
- Use gentle, nature-inspired icons to represent various features and mood states.
- Employ a clean, spacious layout to avoid feeling cluttered or overwhelming.
- Incorporate subtle, calming animations (e.g., gentle fades, smooth transitions) to enhance the user experience without being distracting.
